# Лабораторная работа №3

## Развертывание и эксплуатация дообученной языковой модели

В рамках лабораторной работы была выполнена задача развертывания дообученной языковой модели, полученной в лабораторной работе №2, и организации взаимодействия с ней в режиме инференса различными способами: через консольный интерфейс и через веб-сервер на базе FastAPI.

В качестве основы использовалась языковая модель **Qwen2.5-1.5B-Instruct**, дообученная методом **QLoRA** на специализированном корпусе текстов по тематике нейрокомпьютерных систем. После дообучения LoRA-адаптер был объединён с базовой моделью в единую структуру весов, что позволило использовать модель как стандартную Hugging Face-совместимую архитектуру.

В ходе выполнения работы реализованы:

* консольный интерфейс взаимодействия с моделью;
* API-сервер на базе FastAPI;
* веб-интерфейс чата через браузер;

---

## Структура проекта

### `my_finetuned_model_lab3/`

Каталог с объединённой дообученной моделью. Содержит конфигурационные файлы, токенизатор и веса модели (`model.safetensors`). Используется для выполнения инференса.

### `run_chat.py`

Скрипт консольного взаимодействия с моделью. Позволяет задавать вопросы через терминал и получать ответы с измерением времени генерации и скорости вывода токенов. 

### `api_server.py`

Серверное приложение FastAPI, реализующее REST API для работы с моделью. Принимает запросы пользователя и возвращает сгенерированные ответы вместе с параметрами инференса. 

### `model_utils.py`

Модуль загрузки модели и токенизатора, формирования диалогового контекста и генерации ответов. Также вычисляет latency и throughput при генерации текста. 

### `config.py`

Файл конфигурации параметров генерации модели: путь к модели, максимальная длина ответа, параметры temperature, top-p и штраф повторений. 

### `static/index.html`

Клиентская часть веб-интерфейса чата, обеспечивающая взаимодействие пользователя с API-сервером через браузер.

### `requirements.txt`

Список используемых зависимостей проекта (transformers, torch, fastapi и др.).

## Результат выполнения работы

В результате лабораторной работы реализовано развертывание дообученной языковой модели в локальной среде и обеспечено взаимодействие с ней через консольный интерфейс и веб-приложение. Модель успешно выполняет генерацию ответов по тематике обучающего корпуса и может использоваться как локальный интеллектуальный консультант.
